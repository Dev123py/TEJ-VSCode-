Useful Formatting Tags

Much like when we are doing word processing, we may want to create bold
text, or italicized text, or underlined text. Any and all tags in HTML must
start with a < and end with a > in order to be valid.

We must also ensure tha any open tage has a corresponding closing tag. For
example, if we have a bold tag, <b> we MUST put a </b> after the text we
wamt to be in bold. Note the difference between and . The end tag
always has a / in it before the name of the tag.

 Here is a list of some useful formatting tages and what they do:
  • - The text between the tags is in bold
  • - The text between the tags is in italics
  • - The text between the tags is underlined
  • - The text between the tags is in subscript
  • - The text between the tags is superscript
